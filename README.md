# Blinkit Sales Data Analysis

 End-to-End Data Analysis Project using Python

---

##  Overview

This project performs **Exploratory Data Analysis (EDA)** on a Blinkit dataset to understand patterns in sales, product categories, and outlet performance.
The goal is to transform raw data into structured insights using Python-based data analysis tools.
